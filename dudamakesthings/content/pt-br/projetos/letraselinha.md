O projeto foi criado em Agosto de 2023 como uma página/perfil no Instagram, lá eu comecei a criar imagens para conseguir postar as coisas que eu escrevia.

Falo que são poesias, mas nem sempre elas possuem a estrutura de um poema. Em sua maioria são alguns versos de uma cadeia de pensamento ao qual apenas o sentimento guia o fluxo.

Eu acabei excluindo o perfil do Instagram, mas quero continuar escrevendo alguns versos quando sentir vontade, por isso, resolvi criar um espaço para o projeto aqui.

Para encontrar os escritos, eu vou tentar criar uma série e também utilizar tags e uma categoria para que eles possam ser encontrados. 

Assim que eu conseguir coloco o link do caminho aqui.
